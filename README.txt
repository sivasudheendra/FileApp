
FileApp - This project is used to parse the text file and convert the parsed content into CSV & XML content.

Prerequisites:
Java: JDK 1.8
MAVEN:apache-maven-3.6.3
SPRING BOOT:2.4.0


How to Run the Application:

1) Firstly, Set the propery values in the application.properties file for input file and output file
i.e input.file.name // Input text file for parsing
output.xml.file.name // Output will be generated in xml format
output.csv.file.name // Output will be generated in csv format

2) Log File related changes should be done in log4j.properties file.

3) Run the File FileApp.java

If it run successfully,please check the output folders for the files generated.
